## 💡 与Java对比要点

### 变量声明
- Go: `name := "value"` 更简洁
- Java: `String name = "value"` 必须指定类型

### 类型转换
- Go: 所有转换必须显式 `float64(i)`
- Java: 支持自动向上转型

### 常量
- Go: `const` + `iota` 实现枚举
- Java: `final` 或 `enum`类型

### 字符串
- Go: 字符串是基本类型，不可变
- Java: String是类，也不可变
