## 前言

这不仅仅是一份面试题汇总，更是一本帮助你从底层原理理解 MySQL 的实战指南。

在面试中，面试官往往不满足于“背诵”标准答案，他们更看重你对技术原理的深度理解。例如，当问到“锁”时，他们想听的不是简单的“表锁和行锁”，而是“意向锁解决了什么问题？”、“Next-Key Lock 是如何解决幻读的？”、“死锁是如何检测的？”。

本手册针对这些高频且深度的考点，进行了全面的重写和扩充。我们拒绝枯燥的定义堆砌，而是通过**通俗的语言**、**生动的案例**和**底层的视角**，带你彻底攻克 MySQL 面试难关。
