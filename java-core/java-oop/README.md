## 💡 核心概念

### OOP 三大特性

| 特性 | 说明 | 关键字 | 作用 |
|------|------|--------|------|
| **封装** | 隐藏内部实现细节 | private, public | 保护数据、降低耦合 |
| **继承** | 子类继承父类的特性 | extends, super | 代码复用、建立层次 |
| **多态** | 同一接口多种实现 | @Override | 灵活扩展、统一接口 |

## 🎯 面试高频问题

### 1. 面向对象的三大特性是什么？
- **封装**：隐藏实现细节，通过public方法访问private数据
- **继承**：子类继承父类的属性和方法，实现代码复用
- **多态**：同一方法调用产生不同行为

### 2. 重载（Overload）和重写（Override）的区别？
- **重载**：同一类中方法名相同，参数列表不同
- **重写**：子类重新实现父类方法，方法签名必须相同

### 3. this 和 super 的区别？
- **this**：指向当前对象，用于访问当前类的成员
- **super**：指向父类对象，用于访问父类的成员

### 4. 什么是向上转型和向下转型？
- **向上转型**：子类 → 父类（自动）
- **向下转型**：父类 → 子类（需要强制转换）

### 5. 为什么要使用封装？
- 隐藏实现细节
- 数据验证和保护
- 提高代码可维护性
- 降低耦合度
