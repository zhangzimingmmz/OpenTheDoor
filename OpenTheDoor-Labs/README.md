# OpenTheDoor Labs

This directory contains the code playgrounds and examples for the OpenTheDoor project.

## Structure

- **java/**
    - `auth/`: Authentication and Security related projects (Spring Boot).
    - `java-core/`: Core Java examples and learning materials.
- **go/**
    - `go/`: Go language examples and projects.

## Usage

This folder is intended to be a separate project. Please move this entire `OpenTheDoor-Labs` folder to a new location on your machine and open it as a separate workspace in your IDE.
